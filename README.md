# 📊 Mobile Sales Dashboard – Power BI Project

## 📌 Project Overview

This project analyzes **mobile sales performance** using **Power BI**.
The dashboard provides insights into sales trends, product performance, customer behavior, and regional sales distribution.

The goal of this project is to transform raw sales data into **interactive visual insights** that help businesses make data-driven decisions.

---

## 🎯 Objectives

* Analyze total **sales, profit, and quantity sold**
* Identify **top performing mobile brands and models**
* Track **sales performance by region**
* Understand **monthly and yearly sales trends**
* Create an **interactive dashboard for business insights**

---

## 🛠 Tools & Technologies

* **Power BI**
* **Power Query**
* **DAX**
* **Data Visualization**
